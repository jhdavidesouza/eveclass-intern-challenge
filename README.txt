-------ABOUT IT


#ENG
--------
Project built with NuxtJS, using Vuetify GUI Framework.
It's a decision-maker mini-game, where there's only one right answer to proceed until you finish and become victorious.
You can navigate through the pages at your will, even get back to the start, so you can play again.

It's a challenge made by Eveclass for an intership opportunity.

#PT-BR
--------
Projeto criado com NuxtJS, usando Vuetify GUI Framework.
Trata-se de um mini-game de decisões, onde existe apenas uma resposta correta para continuar.
Você pode navegar entre as páginas à vontade, até mesmo voltar ao início do jogo para começar novamente.

Foi um desafio feito pela Eveclass para uma oportunidade de estágio.
